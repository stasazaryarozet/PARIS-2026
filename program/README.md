# PROGRAM / Программа тура

Все материалы, касающиеся самого тура (не продвижения).

## Структура

```
program/
└── route/            # Маршруты и локации
    ├── curators/    # Информация о кураторах
    └── resources/   # Ресурсы (PDF, карты)
```

## Route

### Основные документы
- `ROUTE_SKELETON.md` — скелет маршрута
- `ALTERNATIVES_DAY3.md` — альтернативы для Дня 3
- `SEASONS_PROPOSAL.md` — предложения по сезонам

### Curators
Информация о кураторах:
- `CURATORS_FACTSHEET.md` — факты о кураторах
- `CURATORS_SUMMARY.md` — краткое резюме

### Resources
Дополнительные ресурсы:
- `Programme - Maison&Objet.pdf` — официальная программа Maison & Objet

---

## Отличие от Campaign

**Campaign (кампания)** — продвижение тура (Instagram, лендинг, стратегия).  
**Program (программа)** — сам тур (маршрут, кураторы, локации).
